# Junior Golf Passport Website

Static first version of Junior Golf Passport, a parent-managed profile and memory site for junior golfers.

## Local Preview

From this folder:

```powershell
python -m http.server 8766 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:8766/
```

## Pages

- `/` - landing page
- `/Kara/` - Kara Walker example passport
- `/Kara/map/` - interactive golfed states map
- `/privacy/` - privacy policy draft
- `/terms/` - terms draft
- `/support/` - support page

## Domain

The `CNAME` file is set to:

```text
juniorgolfpassport.com
```
